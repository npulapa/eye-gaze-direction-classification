# Eye Gaze Direction Classification

Real-time eye gaze direction detection using MediaPipe FaceMesh and OpenCV.

## Classes

- Left
- Right
- Center
- Up

## Technologies

- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Press ESC to exit.
